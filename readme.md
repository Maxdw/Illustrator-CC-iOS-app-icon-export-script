Illustrator CC iOS app icon export script
======================
This script will export your active artboard to all retina and non-retina app icon sizes in .png format. The sizes are obtained from a hard-coded JSON/Javascript array that originates from XCode's Contents.json that is generated when creating a Appicon asset in XCode.

----------

## Download
[Download ZIP](https://github.com/Maxdw/Illustrator-CC-iOS-app-icon-export-script/archive/master.zip)

## Usage
1. Create a square artboard of 1024x1024 pixels (also works with other sizes)
2. Design your app icon
3. Run app.icon.jsx in Illustrator with your app icon artboard active (shortcut âŒ˜+12)
4. Select your output folder
5. Place the .png files in your Xcode's Appicon assets at the appropriate spots
